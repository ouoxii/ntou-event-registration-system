### 提升流程品質或系統品質的措施：

- `Code Review`: 透過GitHub的PR來實踐，merge前須經至少兩人同意。
- `API Testing`: 交付RESTful API前皆需Postman或Httpie確認功能正確。
- `GitHub Flow`: 以功能或debug為單位建立分支。
- `JWT`: 使用Spring Security和access token來保護API。
